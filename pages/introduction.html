LiteGapps is an Custom Gapps for smartphones, tablet, desktop, free and open-source software. by providing complete, stable support and several additional features.

small friendly complete

LiteGapps first release on 12 july 2020 supports android Lollipop Up to Android 13. comes with 5 products namely : LiteGapps, LiteGapps++, LiteGapps Controller, LiteGapps Remover LiteGapps Addon

LiteGapps has system and systemless installation system. using 3 installation modes :

[MAGISK] = Magisk Module Flashable

[RECOVERY] = Recovery Module Flashable

[AUTO] = Magisk/Recovery Module Flashable

LiteGapps is made as light and minimal as possible but we also provide variants to meet the needs of the users

LiteGapps provides many Support,Version,Variant so that users can use LiteGapps according to their devices and needs. you can read the documentation so you don't download wrong !

The beginnings of Litegapps

Litegapps is inspired by my habit of always being simple, for years I was looking for the lightest Gapps and I found them but it's not as light as I thought. actually I don't like GoogleApps because it only takes up RAM, battery and makes your cellphone slow. I only need gapps to sync contacts and download games after that I don't need gapps anymore, so I made this Litegapps! With Litegapps you can uninstall all Gapps if you don't need them from your phone and your life! Because this is systemless.

Unique installer

Litegapps uses a unique installer of which you can :

• Install as magisk module/recovery flashable in one package

• Can install/uninstall in magisk manager/recovery in one package

• Supports many versions of android and architecture in one package with a small fixed size

• can uninstall cleanly in one package (without additional uninstaller)

LiteGapps

LiteGapps is specifically targeted for certain android/architecture versions, which are based on Google Pixel devices. It has 9 variants, namely : Lite,Bakso,Core,Go,User,Basic,Nano,Micro,Pixel.

Support :

Architecture = arm,arm64,x86,x86_64

Android = 5.0 to 13.0

LiteGapps++

LiteGapps++ is a development of regular LiteGapps, with LiteGapps++ you can use it for cross-architectures and android versions in 1 flashable ZIP, while maintaining its small size. It has 3 variants, namely Reguler,LTS,MicroG.

LiteGapps++ is made for those of you who like to change roms or android versions, you can use this for many devices and android versions in one flashable ZIP

Support :

Architecture = arm,arm64

Android = 5.0 to 13.0

LTS

LTS is the development of "litegapps" which allows it to support almost all versions of android and architecture while maintaining its small size.

It's a derivative of "litegapps++" distinguishing it's made for the long term

This is intended for those who rarely update "litegapps" so they can use it for a long time.

Architecture = arm,arm64

Android = 5.0 or later

MicroG

Litegapps++ MicroG is the development of "litegapps" which allows it to support almost all versions of android and architecture while maintaining its small size.

It is a derivative of "litegapps++" which distinguishes it from using MicroG GMS and several other properties, thus consuming less RAM and battery size and usage

This is intended for those who are looking for a lighter version of Litegapps using battery and RAM.

Architecture = arm,arm64

Android = 5.0 or later

LiteGapps Controller

LiteGapps Controller is an additional feature of LiteGapps or LiteGapps++ which aims to fully control litegapps, which contains additional features such as.

• Booster

• Gaming Mode

• Download Package Addon

• Battery saver

• Device Info

• Gms Dose

• Tweaks

• And many others

LiteGapps Remover

On the release of android 12 most custom roms have provided gapps in the default rom, for that we provide a solution to overcome this problem.

LiteGapps Remover provides a solution to remove gapps from the default rom, it also uses Systemless and Non Systemless systems.

Google apps like Google Dialer,Google Messenger,Google webview,Google Contact will be changed to AOSP Apps
